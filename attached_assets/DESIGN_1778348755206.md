# Design System — Pujamostucoche.es

## Paleta de colores

| Nombre | Hex | Uso |
|---|---|---|
| Navy (principal) | `#0B1D3A` | Fondo hero, navbar, secciones oscuras, texto destacado |
| Navy Light | `#132B52` | Hover estados, variantes |
| Navy Dark | `#06101F` | Footer |
| Orange (acento) | `#F47B20` | CTAs, highlights, iconos, badges, precios |
| Orange Dark | `#D66A15` | Hover de botones naranja |
| Orange Light | `#FF9A4D` | Glows, sombras suaves |
| White | `#FFFFFF` | Fondos de sección, texto sobre navy |
| Gray 50 | `#F9FAFB` | Fondos alternativos (secciones pares) |
| Gray 400 | `#9CA3AF` | Texto secundario, precios tachados |
| Gray 600 | `#4B5563` | Cuerpo de texto sobre fondo blanco |

### Regla de uso
- **Navy + Blanco** → secciones de hero, misión, confianza final, footer
- **Blanco + Navy** → secciones de contenido: problema, ejemplo real, cómo funciona
- **Gris claro + Navy** → secciones de apoyo: tabla comparativa, modo IKEA, stock flash
- **Naranja** → siempre para CTAs, precios totales y palabras clave emocionales

## Tipografía

**Fuente:** Inter (Google Fonts)
- Weights usados: 400 (body), 500 (medium), 600 (semibold), 700 (bold), 800 (extrabold), 900 (black)

| Elemento | Tamaño base | Tamaño MD | Weight | Color |
|---|---|---|---|---|
| H1 Hero | 30px (text-3xl) | 60px (text-6xl) | 800 | Blanco |
| H2 Sección | 24px (text-2xl) | 36px (text-4xl) | 800 | Navy |
| H3 Card/Step | 18px (text-lg) | 18px | 700 | Navy |
| Body grande | 18px (text-lg) | 18px | 400 | Gray 700 |
| Body normal | 16px (text-base) | 16px | 400 | Gray 600 |
| Caption/micro | 12px (text-xs) | 12px | 400 | Gray 400 |
| CTA button | 18px (text-lg) | 20px (text-xl) | 700 | Blanco |

### Regla de color en titulares
Los H2 siguen el patrón: **texto navy + highlight naranja** en la parte clave del mensaje.
Ejemplo: `Así de simple. <span orange>Así de distinto.</span>`

## Componentes

### Cards de vehículo (Stock Flash)
- Fondo: blanco con borde `border-gray-100`
- Hover: `shadow-xl transition-shadow`
- Badge: naranja sólido, esquina superior izquierda
- Precio tachado: `line-through text-gray-400`
- Precio final: `text-2xl font-extrabold text-navy`
- CTA: naranja full-width, `rounded-xl`

### Cards de pilares (Misión)
- Fondo: `bg-white/5` sobre navy
- Borde: `border-white/10` → hover `border-orange/30`
- Icono: contenedor `bg-orange/20 rounded-xl` + svg naranja

### Steps (Cómo Funciona)
- Número decorativo: `text-6xl font-extrabold text-orange/10` (posición absoluta)
- Tarjeta: `bg-gray-50 rounded-2xl` con hover shadow y borde naranja
- Icono: `bg-orange rounded-xl` con icono blanco

### Tabla comparativa
- Cabecera "Nosotros": dot naranja + texto navy bold
- Cabecera "Ellos": gris 400
- Filas: alternadas blanco / gris 50
- Iconos de fila: naranja
